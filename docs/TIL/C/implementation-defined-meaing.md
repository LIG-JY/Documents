# implementation-defined

## C 에서 `implementation-defined`의 의미

- 특정 기능이나 동작이 표준에 의해 명확히 정의되지 않고, 각 컴파일러나 라이브러리 구현자가 정의하는 것을 의미함
  - 이러한 구현 정의 동작은 각 구현체의 문서에서 확인할 수 있음
- C언어의 철학 유연성에서 생긴 개념과 이를 표현하기 위한 단어
