# Location Tracking Service Design

## Objective

- 위치 추적 서비스는 주기적으로 유저의 위치 데이터를 수집하 자주 방문하는 위치를 식별

## Service Overview

- 주요 기능:
    - 이동 거리 기반 위치 수집
    - 체류 시간 계산 및 자주 방문하는 위치 도출
    - 유저 정보 업데이트

## Data Flow

- 데이터 수집:
    - 모바일 앱에서 위치 데이터를 수집하여 API 서버로 전송
- 데이터 변환:
    - 모델링 후 결정
    - TODO
- 데이터 저장
